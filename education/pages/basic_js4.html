Задание 1

Написать функцию, преобразующую число в объект. Передавая на вход число от 0 до 999, мы должны получить на выходе объект, в котором в соответствующих свойствах описаны единицы, десятки и сотни. Например, для числа 245 мы должны получить следующий объект: {'единицы': 5, 'десятки': 4, 'сотни': 2}. Если число превышает 999, необходимо выдать соответствующее сообщение с помощью console.log и вернуть пустой объект.

Листинг программы: 

function numToObj(num) {
    num = num.toString()
    if (num > 999) {
        console.log('Число больше 999')
        return {}
    }
    if (num < 0) {
        console.log('Число меньше 0')
        return {}
    }
    return {
        'единицы': num[num.length - 1],
        'десятки': num[num.length - 2],
        'сотни': num[num.length - 3],
    }
}

console.log(numToObj(123))

Задание 2

a) В прошлой теме вы реализовали корзину на базе массивов. Какими объектами можно заменить их элементы?
b) Реализуйте такие объекты.
c) Перенести функционал подсчета корзины на объектно-ориентированную базу.

Листинг программы: 

const Basket = {
    goods: [
      {
        id_product: 123,
        product_name: "Ноутбук",
        price: 45600,
        quantity: 1
      },
      {
        id_product: 456,
        product_name: "Мышка",
        price: 1000,
        quantity: 1
      }
    ],
    countBasketPrice() {
      return this.goods.reduce((totalPrice, cartItem) => totalPrice += cartItem.price, 0);
    }
  }

  console.log(Basket.countBasketPrice())
